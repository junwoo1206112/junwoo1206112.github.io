소개

Unity C# 게임 개발자

저는 Unity와 C#을 주력으로 다양한 장르의 게임을 개발하는 게임 개발자입니다. 멀티플레이 낚시 게임, 텍스트 어드벤처, 클리커 게임, 스크롤 게임 등 다양한 프로젝트를 통해 게임 설계와 개발 전반에 걸친 경험을 쌓아왔습니다.

특히 MultiplayFishingGame에서는 Mirror 네트워킹을 활용한 멀티플레이 구현에 도전했고, Text-Adventure-Game에서는 FSM 상태 머신과 AI 게임 마스터 시스템을 갖춘 텍스트 어드벤처 엔진을 Unity C# 기반으로 직접 제작했습니다.

또한 SnowmanCount와 CaterpillarsClickerGame01과 같은 캐주얼 게임부터 Designing-a-Worldview-Simulation-Agent와 같은 Python 기반 AI 세계관 시뮬레이션까지, 폭넓은 개발 스펙트럼을 가지고 있습니다. 게임 개발뿐 아니라 WebBoard와 같은 웹 프로젝트를 통해 JavaScript 기반 개발 경험도 갖추고 있습니다.

새로운 장르와 기술에 도전하는 것을 즐기며, 지속적으로 학습하고 성장하는 개발자가 되기 위해 노력하고 있습니다.

주요 프로젝트

MultiplayFishingGame — Unity, C#, Mirror Networking — 멀티플레이 낚시 게임

Text-Adventure-Game — Unity, C# — FSM 기반 텍스트 어드벤처 엔진

SnowmanCount — Unity, C# — 광고형 캐주얼 게임

CaterpillarsClickerGame01 — Unity, C# — 클리커 게임

Designing-a-Worldview-Simulation-Agent — Python — AI 세계관 시뮬레이션

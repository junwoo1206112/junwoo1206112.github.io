About

문제를 재현하고, 기록하고, 개선하는 개발자

개발 방향

단순히 화면을 만드는 것보다 실제 산업 현장에서 필요한 흐름을 코드로 재현하는 데 관심이 있습니다. 장비 상태를 시뮬레이션하고, 검사 결과를 CSV로 남기고, 모델 결과를 지표로 검증하는 방식으로 프로젝트를 정리하고 있습니다.

대표 역량

제조 소프트웨어

C# 기반 장비 제어 시뮬레이션, 운전 로그, 알람 상태, PASS/FAIL 판정 흐름을 구현했습니다.

비전·AI

Python 기반 결함 이미지 생성, 특징 추출, 분류 모델 학습, 혼동행렬 리포트를 구성했습니다.

검증 습관

README, 실행 방법, 테스트 결과, 샘플 로그를 남겨 채용 담당자가 결과를 빠르게 확인할 수 있게 정리합니다.
